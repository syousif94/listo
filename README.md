## Listo - The Smart List Maker

Listo is an Expo app with a Bun backend. It uses Groq for Kimi inference and Apple for Auth and List Synchronization.

### System Dependencies

Bun bun.sh
expo expo.dev

### Install Local Dependencies

#### App

`bunx expo install`

#### Server

`cd backend && bun i`

### Running

#### App

`bunx expo start`

#### Server

`cd backend && bun dev`
